== README

QnA application built with Ruby on Rails for training course of online web development school Thinknetica. 

Some technologies used in this project:

* Delayed Jobs and scheduling with Sidekiq

* Full-text search with Sphinx

* Pub/sub service with Faye

* Gentleman's tool kit: Devise, omniAuth, CarrierWave, CanCanCan, etc.

* REST API

* Monitoring with monit

* OAuth provider with doorkeeper
